### Full Description (4000 characters max)
```
SmokeSpot은 전국의 흡연구역과 금연구역 정보를 제공하는 지도 앱입니다.

주요 기능:
- 전국 66,000개 이상의 흡연구역/금연구역 위치 정보
- 현재 위치 기반 주변 흡연구역 검색
- 실시간 지도에서 흡연/금연구역 확인
- 즐겨찾기 기능으로 자주 가는 장소 저장
- 사용자 직접 등록 기능
- 사진이 포함된 상세 정보 제공
- 영업시간, 편의시설(지붕, 의자) 정보

데이터 출처:
- 공공데이터포털 (data.go.kr) 전국 금연구역 표준데이터
- 세종시 흡연구역/금연구역 API
- 지자체별 공공데이터 API
- Kitsuenjo 크라우드소싱 데이터

사용 시나리오:
- 외출 전 근처 흡연구역 미리 확인
- 낯선 지역에서 빠르게 흡연구역 찾기
- 금연구역 확인으로 과태료 방지
- 편의시설이 좋은 흡연구역 찾기

앱 특징:
- 아름다운 오로라 테마의 UI 디자인
- 빠른 검색과 필터링 기능
- 오프라인 지원 (즐겨찾기 데이터)
- 개인정보 보호 (모든 데이터 기기 내 저장)

참고사항:
- 위치 정보는 앱 사용 중에만 수집됩니다
- 공공데이터 API 연동을 위해 인터넷 연결이 필요합니다
- 일부 데이터는 최신 정보와 다를 수 있습니다

문의: [개발자 이메일]
```
